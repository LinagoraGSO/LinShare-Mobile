------ LinShare Mobile ------

* Téléchargement des outils :

	1. Afin de faire fonctionner l'application, il faut utiliser un module d'Eclipse propre au développement Android
		-> téléchargement : http://developer.android.com/sdk/index.html
	2. Une fois téléchargé, dézipper l'archive.
	3. Pour ouvrir Eclipse 
		-> adt-bundle-"votre version" > eclipse > eclipse
		Comme indiqué sur le site, le module est déjà préchargé et prêt à l'emploi.
	
* Ouverture du projet :

	1. Récupérer le dossier LinShare Mobile contenant l'intégralité des fichiers nécessaires au fonctionnement de l'application.
	2. Déplacer ce dossier à l'endroit que vous voulez utiliser comme Workspace.
	3. Lancer Eclipse et choisir le dossier où vous avez déplacer LinShare Mobile comme Workspace.
	4. Sous Eclipse : File > New > Project.
	5. Sélectionner General > Project
	6. Project name : "LinShare Mobile", il est important de spécifier le même nom que celui du projet initial afin que la copie puisse s'opérer.
	7. Finish

A cette étape, le projet est prêt à fonctionner, il suffit de cliquer sur l'îcone de lancement "Run" pour que le projet se lance.

* Type de lancement :

	o Sur mobile physique :

		1. Pour Android, il est nécessaire au préalable d'activer l'option Paramètres > Options de développement > Débogage USB.
		2. Connectez le câble micro-USB au portable.
		3. L'appareil apparaîtra dans la liste des plateformes disponibles pour lancer le programme une fois que vous aurez cliquer sur l'icône "Run".
		
	o Sur émulateur :
	
		1. Suivez le tutoriel suivant qui résumé comment créer son propre émulateur
			-> http://www.siteduzero.com/informatique/tutoriels/creez-des-applications-pour-android/l-emulateur-de-telephone-android-virtual-device
		2. De même que pour un mobile physique, l'émulateur apparaîtra dans les supports disponibles.
